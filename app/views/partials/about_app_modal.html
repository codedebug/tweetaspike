About Tweetaspike

This sample application is built using the ASEAN Stack — AeroSpike + Express + AngularJS + Node.js. Two Aerospike's open sourced technologies being demonstrated in this app are:
  • Aerospike's In-Memory NoSQL Database
  • Aerospike's Node.js Client

To learn more about Aerospike's technologies, please click here. For questions/comments about this app or if you'd like to know how you can write your own app leveraging Aerospike's technologies, feel free to drop me a line at dash @ aerospike dot com.

Application Architecture:

  • In-Memory NoSQL Database Server — Amazon EC2 m1.small AMD64 instance running Ubuntu 12.04; Image Size 8GB
  • Web application — Amazon EC2 m1.small instance running Amazon Linux mounted with Node.js EBS; Image Size 8GB

Application Features:

  • Register | Login | Logout
  • Post — similar to tweets only better since there's no character limit :)
  • Follow — follow other tweetaspikers
  • Unfollow — unfollow tweetaspikers you are following
  • Following — list of tweetaspikers you follow including their posts
  • Followers — list of tweetaspikers that follow you including their posts
  • Alerts — receive real-time alerts when tweetaspikers you are following add new posts

Other technologies that I've used in this app are: Angular UI | Socket.io | Bower | Yeoman | Grunt
