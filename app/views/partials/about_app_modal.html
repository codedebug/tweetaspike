About Tweetaspike

This sample application has been built using the ASEAN Stack -- AeroSpike + Express + AngularJS + Node.js. Two Aerospike's open sourced technologies being demonstrated in this app are:

  • In-Memory NoSQL Database
  • Node.js Client

To learn more about Aerospike's technologies, please click here. For questions or comments about this app, feel free to drop me a line at dash @ aerospike dot com.

Other technologies being used in this app worth mentioning are: Angular UI | Socket.io | Bower | Yeoman | Grunt

Application Features:

  • Register
  • Login
  • Post -- similar to tweets only better since there's no character limit :)
  • Follow -- follow other tweetaspikers
  • Unfollow -- unfollow tweetaspikers you are following
  • Following -- list of tweetaspikers you follow including their posts
  • Followers -- list of tweetaspikers that follow you including their posts
  • Alerts -- receive real-time alerts when tweetaspikers you are following add new posts
  • Logout
